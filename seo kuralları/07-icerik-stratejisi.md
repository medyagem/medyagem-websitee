# İçerik Stratejisi Rehberi

## İçerik Stratejisi Nedir?

İçerik stratejisi, hedef kitlenize değer sağlayan, SEO hedeflerinizi destekleyen ve işletme amaçlarınıza hizmet eden içeriklerin planlanması, oluşturulması ve yönetilmesi sürecidir.

> "İçerik kraldır." - Bu SEO dünyasının en bilinen sözüdür çünkü kaliteli içerik olmadan sıralama mümkün değildir.

## İçerik Stratejisinin Temelleri

### 1. Kullanıcı Niyetini Anlama

#### Niyet Türleri ve İçerik Eşleştirmesi

| Niyet Türü | Kullanıcı Ne İstiyor? | İçerik Türü |
|------------|----------------------|-------------|
| Bilgilendirici | Bilgi öğrenmek | Blog, rehber, SSS |
| Navigasyonel | Belirli siteye ulaşmak | Ana sayfa, hakkımızda |
| Ticari | Araştırma/karşılaştırma | Karşılaştırma, inceleme |
| İşlemsel | Satın alma/hizmet alma | Ürün/hizmet sayfası |

#### Niyet Belirleme Yöntemi
```
Anahtar kelime: "su kaçağı"

SERP Analizi:
- Üst sonuçlar bilgilendirici içerik mi? → Bilgilendirici niyet
- Üst sonuçlar hizmet sayfaları mı? → İşlemsel niyet
- Karışık sonuçlar mı? → Karma niyet (her iki türde içerik üret)
```

## İçerik Türleri

### 1. Evergreen İçerik
Zamanla değerini kaybetmeyen içerikler.

**Örnekler:**
- "Su Kaçağı Nasıl Tespit Edilir?" (rehber)
- "Tıkanıklık Açma Yöntemleri" (how-to)
- "Tesisat Terimleri Sözlüğü" (kaynak)

**Özellikleri:**
- Uzun ömürlü trafik
- Link çekme potansiyeli yüksek
- Düzenli güncelleme gerektirir

### 2. Güncel İçerik
Zamana duyarlı, trend konular.

**Örnekler:**
- "2025 İSKİ Tesisat Fiyatları"
- "Kış Aylarında Boru Donması Önleme"
- "Yeni Tesisat Yönetmeliği Değişiklikleri"

**Özellikleri:**
- Hızlı trafik spike'ı
- Kısa ömürlü
- Sosyal medya paylaşım potansiyeli

### 3. Hizmet/Ürün Sayfaları
Dönüşüm odaklı sayfalar.

**Yapısı:**
```
1. Başlık (H1) - Anahtar kelime + Değer önerisi
2. Hero section - Ana mesaj + CTA
3. Sorun tanımı - Müşterinin acısı
4. Çözüm sunumu - Hizmet detayları
5. Özellikler/Avantajlar
6. Sosyal kanıt (yorumlar, referanslar)
7. SSS bölümü
8. Son CTA
```

## İçerik Optimizasyonu

### On-Page SEO Faktörleri

#### Title Tag (Başlık Etiketi)
```
Format: [Birincil Anahtar Kelime] - [İkincil/Marka] | [Site Adı]
Uzunluk: 50-60 karakter

Örnek:
"Su Kaçağı Tespiti İstanbul - 7/24 Acil Servis | Berat Tesisat"
```

#### Meta Description
```
Format: [Sorun] + [Çözüm] + [CTA]
Uzunluk: 150-160 karakter

Örnek:
"İstanbul'da su kaçağı mı var? Kameralı tespit sistemiyle kaynağı 
30 dakikada buluyoruz. 7/24 acil servis. Hemen arayın: 0534 383 7758"
```

#### Başlık Hiyerarşisi
```html
<h1>Su Kaçağı Tespiti İstanbul</h1>
  <h2>Su Kaçağı Belirtileri</h2>
    <h3>Görünür Belirtiler</h3>
    <h3>Gizli Belirtiler</h3>
  <h2>Tespit Yöntemlerimiz</h2>
    <h3>Kameralı Tespit</h3>
    <h3>Termal Tespit</h3>
  <h2>Hizmet Bölgelerimiz</h2>
  <h2>Sık Sorulan Sorular</h2>
```

#### İçerik Uzunluğu Rehberi
| İçerik Türü | Önerilen Uzunluk |
|-------------|------------------|
| Blog yazısı | 1.500-2.500 kelime |
| Hizmet sayfası | 800-1.500 kelime |
| Pillar page | 3.000-5.000 kelime |
| Ürün açıklaması | 300-500 kelime |
| SSS cevabı | 50-200 kelime |

### Görsel Optimizasyonu

#### Görsel SEO Checklist
```
□ Dosya adı optimize (su-kacagi-tespiti.jpg)
□ Alt text eklendi (açıklayıcı, anahtar kelimeli)
□ Boyut sıkıştırıldı (WebP format)
□ Boyutlar belirtildi (width, height)
□ Lazy loading aktif
□ Responsive görseller kullanıldı
```

#### Alt Text Örnekleri
```html
<!-- KÖTÜ -->
<img src="img1.jpg" alt="resim">
<img src="foto.jpg" alt="su kaçağı su kaçağı tespiti kaçak su">

<!-- İYİ -->
<img src="su-kacagi-tespiti-termal-kamera.jpg" 
     alt="Termal kamera ile duvardaki su kaçağı tespiti işlemi">
```

### İç Link Stratejisi

#### İç Link Kuralları
1. **Doğal yerleştir** - Kullanıcı deneyimini düşün
2. **Alakalı sayfalara link ver** - Konusal ilişki önemli
3. **Açıklayıcı anchor text** kullan
4. **Önemli sayfalara daha çok link** ver
5. **Orphan sayfa bırakma** - Her sayfa en az 1 iç link alsın

#### İç Link Yapısı Örneği
```
/su-tesisati-rehberi/ (Pillar)
    ↓ link verir
/su-kacagi-tespiti/ (Cluster)
    ↓ link verir
/blog/su-kacagi-belirtileri/ (Supporting)
    ↓ link verir
/bolge/atasehir-su-tesisatcisi/ (Location)
```

## İçerik Kalite Kriterleri

### 10x Content Framework
İçeriğiniz rakiplerden 10 kat daha iyi olmalı:

```
KONTROL LİSTESİ:
□ Daha kapsamlı mı?
□ Daha güncel mi?
□ Daha iyi yapılandırılmış mı?
□ Daha iyi görselleştirilmiş mi?
□ Orijinal veri/araştırma içeriyor mu?
□ Daha iyi UX sunuyor mu?
□ Daha hızlı yükleniyor mu?
□ Mobilde daha iyi görünüyor mu?
□ Daha güvenilir kaynaklar kullanıyor mu?
□ Daha pratik/uygulanabilir mi?
```

### Content Quality Signals (Google'ın Baktıkları)

#### contentEffort (İçerik Çabası)
Google, içeriğe harcanan çabayı ölçer:
- Orijinal araştırma
- Uzman görüşleri
- Özel görseller
- Detaylı analiz
- Benzersiz bakış açısı

#### originalContentScore (Orijinallik)
- Kopya içerik cezalandırılır
- Spin edilmiş içerik tespit edilir
- Benzersiz değer aranır

## İçerik Güncelleme Stratejisi

### Güncelleme Gerektiren Durumlar
1. **Bilgi eskimesi** - Tarihler, fiyatlar, yasalar
2. **Sıralama düşüşü** - Performans kaybeden sayfalar
3. **Yeni bilgi** - Sektörel gelişmeler
4. **Rakip geliştirme** - Rakipler daha iyi içerik ürettiyse

### Güncelleme Süreci
```
1. Performans analizi
   └── Search Console'da düşen sayfaları bul

2. Rakip analizi
   └── Sıralayan rakipleri incele

3. İçerik genişletme
   └── Eksik konuları ekle

4. Bilgi güncelleme
   └── Eski verileri yenile

5. Görsel ekleme
   └── Yeni görseller, infografikler

6. Tarih güncelleme
   └── "Son güncelleme" tarihini değiştir

7. Yeniden yayınlama
   └── Sosyal medyada paylaş
```
