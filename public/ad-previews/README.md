# Google Ads Önizleme Görselleri

Bu klasöre Google Ads reklam önizleme görsellerinizi ekleyin.

## Dosya Adlandırma

Görselleri şu şekilde adlandırın:
- `ad1.png` veya `ad1.jpg`
- `ad2.png` veya `ad2.jpg`
- `ad3.png` veya `ad3.jpg`
- `ad4.png` veya `ad4.jpg`

## Önerilen Format

- **Format:** PNG veya JPG
- **Boyut:** Mobil ekran görüntüsü boyutunda (yaklaşık 380x640px veya benzer)
- **Aspect Ratio:** 9:16 (mobil telefon oranı)

## Nasıl Eklenir?

1. Google Ads hesabınızdan reklam önizlemelerini ekran görüntüsü olarak kaydedin
2. Görselleri bu klasöre kopyalayın
3. Dosya adlarını `ad1.png`, `ad2.png` şeklinde düzenleyin
4. Daha fazla görsel eklemek için `components/AdShowcase.tsx` dosyasındaki `adPreviews` array'ini güncelleyin

## Not

Görseller yüklenene kadar placeholder gösterilecektir.
